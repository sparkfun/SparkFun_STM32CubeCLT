# SparkFun STM32CubeCLT

This repo contains SparkFun's copy of ST's STM32CubeCLT command-line toolset for Linux

Because STM32CubeCLT is licenced software, the copy here is passphrase encrypted

Due to its size, the st-stm32cubeclt zip is stored with Git LFS (Large File Storage)

You can of course download your own copy of [STM32CubeCLT](https://www.st.com/en/development-tools/stm32cubeclt.html) from ST - provided you login and agree to the terms of [the licence](https://www.st.com/resource/en/license_agreement/dm00218346.pdf)
